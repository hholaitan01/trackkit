HOW THIS WORKS

// 1. Get GPS from browser navigator.geolocation.watchPosition(pos => { // 2. Send to TrackKit API every 5s fetch(`${API}/v1/deliveries/${id}/location`, { method: "POST", headers: { "Authorization": `Bearer ${apiKey}`, "Content-Type": "application/json" }, body: JSON.stringify({ lat: pos.coords.latitude, lng: pos.coords.longitude, speed: pos.coords.speed, heading: pos.coords.heading }) }) })
That's it. The API auto-calculates ETA, broadcasts location via WebSocket to all tracking widget subscribers, and triggers status transitions (e.g. DELIVERING → ARRIVING when <200m away).
